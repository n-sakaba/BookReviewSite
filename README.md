# BookReviewSite
本のレビューサイト

## ■ URL
https://n-sakaba.github.io/BookReviewSite/

---

## ■ 概要
- 本のレビューを投稿・管理できるWebアプリ
- レビューの作成・編集や公開/非公開の切り替えができる
- シンプルなUIで、自分だけのレビュー管理とお気に入り整理ができるのが特徴

---

## ■ 使用技術

### フロントエンド
- React
- TypeScript
- CSS

### バックエンド / DB
- Supabase（PostgreSQL）

### その他
- Supabase Auth（認証機能）

---

## ■ 機能一覧

- ユーザー登録・ログイン機能（Supabase Auth）
- レビュー投稿機能（作成・編集・削除）
- レビューの公開 / 非公開切り替え機能
- レビュー閲覧機能（公開レビューの一覧表示）
- お気に入り機能（レビューの保存）

---

## ■ 非機能一覧

### テスト
（※実施していれば記載）
- React Testing Library：コンポーネントテスト

### その他
- 認証管理：Supabase Authによるセッション管理
- データ管理：SupabaseによるリアルタイムDB

---

## ■ こだわりポイント

### ① 公開 / 非公開機能の実装
ユーザーがレビューを「公開・非公開」で切り替えられるようにし、
自分用メモとしても使える設計にしました。

### ② シンプルで使いやすいUI
レビュー投稿・編集の導線をシンプルにし、
直感的に操作できるUIを意識しました。

### ③ Supabaseを活用した認証・データ管理
バックエンドにSupabaseを採用することで、
認証・DB・APIを一元管理し、開発効率と保守性を高めました。
